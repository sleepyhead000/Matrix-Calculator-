# Matrix Calculator

Simple browser-based matrix calculator that shows full step-by-step solutions.

## Run locally

1. Open `index.html` in any browser.
2. Or open `main.htm` (legacy entry file kept for compatibility).

## Deploy on Vercel (Public URL)

### Option A: Vercel dashboard (easiest)

1. Push this repo to GitHub.
2. Go to Vercel and click `Add New -> Project`.
3. Import the repository.
4. Framework preset: `Other`.
5. Build command: leave empty.
6. Output directory: leave empty.
7. Click `Deploy`.

Vercel will serve `index.html` as a static site.

### Option B: Vercel CLI

1. Install Vercel CLI:
   `npm i -g vercel`
2. Inside this folder, run:
   `vercel`
3. For production deployment:
   `vercel --prod`
